# Inova CRM AI

Plataforma CRM SaaS da Inova TI — omnichannel (Chatwoot), automação (n8n), event-driven (Redis + RabbitMQ + Workers) e IA.

## Status

Fundação do repositório. Implementação conforme Plano Mestre e Spec Kit.

## Stack prevista

- Frontend: Next.js, TypeScript, Tailwind, shadcn/ui
- Backend: NestJS, Prisma, PostgreSQL
- Mensageria: Redis, RabbitMQ, Workers
- IA: FastAPI, RAG, agentes especializados
- Integrações: Chatwoot e n8n dedicados
- Infra: Docker Compose, Cloudflare Tunnel, VPS Hetzner

## Domínio

- App: `crm.inovatitech.com.br`
